# 🔖 React Chatting

> React + Redux + Firebase + Bootstrap
>
> + 👋 **TEST SITE** -> https://react-chatting-79415.web.app/

## 💻 기능

> 간단한 채팅 앱

1. 채팅방 생성

   > 채팅방 셍, 즐겨찾기, 채팅 전송, 이미지 전송, 타이핑 입력 체크, 메세지 검색

2. 개인 메세지

   > 위와 채팅 기능 동일
              
2. 회원정보

    > 회원가입, 로그인, 프로필 이미지 변경
